BioJava Webservices

BioJava is a powerful library of bioinformatics tools, but leveraging BioJava from web interfaces is challenging. This provides a number of APIs and pre-build interfaces for accessing BioJava through the browser.

biojava-structure

BioJava understands and can automatically fetch a wide variety of structure formats. In the following examples, :id can be replaced by any of the following:

  • PDB PDB identifier, optionally followed by chain and/or residue ranges. Examples: 4hhb, 4hhb.A, 4hhb.A:1-50.
  • SCOP SCOPe domain. Example: d1h6w.2
  • PDP Protein Domain Parser domain. Example: PDP:4HHBAa
  • CATH Cath domains. Example: 1qvrC03
  • ECOD ECOD domain. Example: e1lyw.1
  • BIO Biological assembly. Example: BIO:2ehz:1

Fetching structures

/pdb/:id
 Example: /pdb/1itb.A 
/mmcif/:id
 Example: /mmcif/1itb.A
Returns the requested structure in PDB or mmCIF format.

Viewing structures

/ngl/:id
 Example: /ngl/1itb.A
Views the requested structure using NGL.

CE-Symm

/cesymm/:id
 Example: /cesymm/1itb.A
Analyze the structure for internal symmetry using CE-Symm.

Information about the alignment is available in several formats: 

/cesymm/:id/pdb
 Example: /cesymm/1itb.A/pdb
PDB file with a superposition of the structure. 
/cesymm/:id/json
 Example: /cesymm/1itb.A/json
JSON describing the alignment.
